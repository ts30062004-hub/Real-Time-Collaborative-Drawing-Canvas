#  Real-Time Collaborative Drawing Canvas

A real-time, multi-user drawing application where multiple users can draw, erase, and add shapes or text simultaneously â€” with live synchronization through WebSockets.

##  Features
- Real-time multi-user drawing
- Brush, eraser, shapes, and text tools
- Undo/Redo synchronized for all users
- Clear canvas and user count display

##  Tech Stack
| Layer | Technology |
|--------|-------------|
| Frontend | HTML5, CSS, JavaScript |
| Backend | Node.js + Express + Socket.io |
| Communication | WebSockets |

##  Setup
1. `cd server && npm install`
2. Run with `node server.js`
3. Open `localhost:3000` in browser
4. For multiple users again open through `localhost:3000`
